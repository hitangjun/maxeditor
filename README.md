# Max Editor

最大化编辑器，隐藏侧边栏和面板，获得更大的代码编辑空间。

## 功能

- **最大化编辑器**：隐藏左侧边栏、右侧边栏和底部面板
- **恢复布局**：再次执行命令恢复之前的布局

## 使用方法

### 快捷键

- **最大化/恢复**：`Ctrl+Shift+M` (Windows/Linux) 或 `Cmd+Shift+M` (Mac)

### 命令面板

1. 打开命令面板 (`Ctrl+Shift+P`)
2. 输入 `Max Editor`
3. 选择 `最大化/恢复编辑器`

### 右键菜单

- 编辑器标签右键 → `Max Editor: 最大化/恢复编辑器`
- 文件资源管理器右键 → `Max Editor: 最大化/恢复编辑器`
